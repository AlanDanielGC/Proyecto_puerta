# Proyecto_puerta
Este proyecto tiene por finalidad crear y lanzar un programa que ayude a la gestión de accesos al centro de computo ubicado en el instituto tecnológico de Cuautla.

# Lenguaje a usar #
*React
*Pyhton

# Base de datos #

# comandos para iniciar el programa #

Para inicar el programa debemos de lanzar los comandos:

1. cd Puerta_IoT
2. npm install
3. npm run start

Con eso iniciaremos el programa.
Para correr la version web lanzaremos: npm run web
Para correr la version de android se debera descargar la aplicacion "Expo Go" en el celular y escanear el Codigo QR que te dara en la terminal, despues lanzar el comando: npm run android
